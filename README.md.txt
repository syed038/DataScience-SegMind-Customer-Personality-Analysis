# Customer Personality Analysis & Targeted Marketing (Clustering Project)

This project performs customer segmentation using KMeans clustering based on demographics and spending behavior. It helps businesses target the right customers and personalize marketing campaigns.

📊 Dataset
- Source: [Kaggle - Customer Personality Analysis](https://www.kaggle.com/datasets/imakash3011/customer-personality-analysis)
- Contains data on 2,200+ customers with features like Age, Income, Marital Status, Product Spending, and more.

💡 What I Did
- Cleaned and preprocessed the data
- Engineered new features like `Age` and `Total_Spent`
- Used the **Elbow method** to find the optimal number of clusters
- Applied **KMeans** to segment customers
- Visualized customer clusters by income and spending
- Extracted actionable business insights

🛠️ Tools Used
- Python, Pandas, Seaborn, Matplotlib
- Scikit-learn
- Jupyter Notebook

📌 Key Skills Demonstrated
- Data Cleaning
- Feature Engineering
- Clustering & Machine Learning
- Business Insight & Visualization

📎 Output
You can see customer clusters plotted by income and total spending using KMeans.

🚀 How to Run
1. Clone this repo
2. Open `customer_analysis.ipynb` in Jupyter
3. Run all cells
